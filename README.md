# rewind-me / research

This repository is updated automatically.
It contains the library articles that have been released so far as part of the [rewind-me](https://rewind-me.com) project.

Rewind-Me is a catalog of human knowledge — not a museum, not a textbook. A working document that walks through history one day at a time: from stone tools to semiconductors, from fire to nuclear reactors, from bone needles to the internet. Every entry documents what a thing is, where it comes from, how to make it, and what it means if you ever had to start from scratch.

The knowledge here is not locked behind a platform or a subscription. It belongs to the species. It always did.
